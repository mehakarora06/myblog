##  Features

✅ User Authentication (Signup/Login/Logout)  
✅ Create, Read, Update, Delete (CRUD) Posts  
✅ Author-based Access Control (Only owner can edit/delete)  
✅ Responsive UI using Bootstrap  
✅ Clean and user-friendly interface  

---

## Project Description

This project is a blogging platform that allows users to register, log in, and create their own blog posts. Each post is linked to a specific user, ensuring proper ownership and security.

The application demonstrates real-world concepts like authentication, authorization, and database management using Django.

---

##  Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- Hosting: PythonAnywhere

---

##  Security Features

- Only authenticated users can create posts  
- Only post owners can edit or delete their posts  
- Prevents unauthorized access
